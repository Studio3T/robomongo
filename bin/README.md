Bash fronted for CMake
======================

If you want to use this scripts, you need to set single environment variable:

    export ROBOMONGO_CMAKE_PREFIX_PATH="/path/to/qt-5/5.5/gcc_64;/path/to/robomongo-shell"

Build Robomongo:

    $ bin/configure
    $ bin/build
    
Install Robomongo:

    $ bin/install
    
Pack Robomongo:

    $ bin/pack
